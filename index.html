    Welcome to Vanilla

    Vanilla is a super awesome extension of LaTex that aims to reduce the learning curve of LaTex and reduce verbosity in the LaTex code. It enhances a lot of features available in LaTex and adds features on top of it to enhance productivity of LaTex users. It was greatly inspired by Markdown, Less CSS, Ruby and Matlab. Read the documentation to know more about Vanilla.

    Technical Details

    Vanilla compiler was originally conceived and written in Python. It was then translated into Ruby. The compiler is built on top of PDFLatex and PDFLatex is a dependency for Vanilla. The command line utility is aimed towards Windows platform but technically it is a cross platform product. Please read the documentation to know more about the internal workings of the compiler. A debugger and error corrector are being written and will be released in Version 2.
