Welcome to Vanilla

Vanilla is a small markup language that is strictly a superset of the LaTex typesetting system. It tries to remove the barriers for people to get the benefits of LaTex.It greatly simplifies several features that are available in LaTex and adds more features on top to increase the productivity of the users. It brings several ideas of Markdown to LaTex and also adds ideas inspired from Less CSS, Matlab and Ruby. It is completely interoperable and backwards compatible with LaTex. It is an ongoing project. So please check back for more updates.

Technical Details

Vanilla compiler code named "Hippo" is written in Ruby. It is a command line utility that converts .vaml(Vanilla extension) into .tex(LaTex extension). It is available in both windows and mac environments. Vanilla provides Ruby as an embedded scripting language for user's programming needs.
